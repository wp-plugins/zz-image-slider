Responsive image slider.

== Description ==

ZZ Image Slider is a WordPress plugin bassed on JSSOR Slider. This plugin is perfectly suitable for homepage and also can be ussed in WordPress posts or pages for showing awesome responsive images slideshows.

= Features: =

* Responsive design
* Auto slide show
* Navigation bullets
* Navigation arrows
* Custom slides timings
* Pause on mouse hover
* Custom each slide title, description and "Read More" link
* Drag and drop to change image place

Samples can be found [here](http://zzapps.net/zzis/zzis-samples/).

== Installation ==

= From your WordPress dashboard =

1. Visit "Plugins > Add New"
1. Search for "ZZ Image Slider"
1. Activate "ZZ Image Slider" from your Plugins page.
1. Visit "ZZ Image Slider > Add New Image Slider" to create new slider.
1. Copy and paste shortcode of the slider to the post or page.

= From WordPress.org =

1. Download "ZZ Image Slider"
1. Upload the "zz-image-slider" directory to your "/wp-content/plugins/" directory, using your favorite method (ftp, sftp, scp, etcâ€¦)
1. Activate "ZZ Image Slider" from your Plugins page.
1. Visit "ZZ Image Slider > Add New Image Slider" to create new slider.
1. Copy and paste shortcode of the slider to the post or page.
